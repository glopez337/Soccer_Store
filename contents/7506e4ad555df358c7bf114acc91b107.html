Cadastrar Produto

UMLUseCase
Projeto Modelo Soft :: Caso de uso :: Cadastrar Produto

Description

Fluxo Principal: O use case inicia quando o Fornecedor se autentica no sistema, Fornecedor clica para cadastrar seu produto, e preenche com as informações do mesmo. Sistema salva as informações do produto, e exibe na página.

Fluxo Excepcional: Se o Fornecedor preenche com informações inválidas, o use case reinicia.
